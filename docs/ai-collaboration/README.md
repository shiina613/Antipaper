# AI collaboration evidence

Thư mục này chứa bằng chứng được tham chiếu bởi [`docs/AI_COLLABORATION_LOG.md`](../AI_COLLABORATION_LOG.md).

```text
ai-collaboration/
├── sessions/      # Markdown, JSON, JSONL, HTML hoặc PDF export từ công cụ AI
└── screenshots/   # PNG/JPG thể hiện phiên làm việc liên quan
```

Quy ước tên file:

```text
AI-<số>-<yyyy-mm-dd>-<công-cụ>-<mô-tả-ngắn>.<ext>
```

Ví dụ:

```text
AI-001-2026-07-18-codex-create-collaboration-log.md
AI-001-2026-07-18-codex-create-collaboration-log.png
```

Không lưu credential hoặc dữ liệu nhạy cảm. Nếu screenshot đã được che dữ liệu, thêm hậu tố `-redacted` và ghi chú trong collaboration log.
